XML Morph

UI / Overview — unified workspace for transforming and structuring XML content, with streamlined controls that simplify editing, formatting, and output management
xml tree toggle validation
Watch on YouTube

Main functions

  • Initial storage
    stores the user's original pasted content so it can be reverted or compared later
  • HTML to XML conversion
    converts HTML into a custom XML format that includes structural elements like chapters, appendices, clauses, and sections. it also adds headers, handles special characters, and replaces standard list tags with more semantic equivalents
  • XML to HTML conversion
    reverses the above process, stripping out XML wrappers and reconstructing readable HTML
  • List handling
    distinguishes between clause-style lists and normal lists to ensure only the appropriate ones are converted to <clause> elements
  • Section wrapping
    automatically groups headings and their following content into semantic sections based on heading levels
  • Preview and error display
    renders color-coded structured output and displays error messages or a parsed view as needed
  • Cleanup
    removes style elements and ensures self-closing tags are handled correctly
  • Validation
    checks for matching and balanced HTML/XML tags, reporting issues if any tags are unbalanced or improperly structured

CodeBOT

2020 Document Conversion & Accessibility

Analytics-driven automated HTML editor that opens Word documents of all sizes while applying the Canadian Standard on Web Accessibility. CodeBOT combines a sequential processing pipeline from core syntax cleanup to content structuring, streamlining the conversion workflow and supporting WCAG compliance.

UI / Overview — main editor workspace with real-time diagnostics, syntax & accessibility validation, and table accessibility settings in a single interface
link validation report system architecture

Modules added since rollout

  • TableNote Pro (2022): an enhanced version of TableNote (2021), a tool for auto-generating footnotes in HTML tables. It provides new options such as toggling footnotes on or off, visual highlighting to identify and trace footnote references, and the ability to inject the footnote section directly within the table
  • Link Validator (2024): a utility that runs in parallel yet independently of main editing tasks. It scans all hyperlinks to detect broken, redirected, or malformed URLs and generates a detailed status report complete with visual indicators, copy/export tools, and an HTTP status legend. It features a 3-slot FIFO popup architecture ensuring only the latest reports are visible. Built from scratch, this component can be integrated into other systems with minimal modification
  • Paste from external sources (2025): this feature expands CodeBOT’s capabilities beyond its original support for .docx files and plain text, allowing users to copy and paste content from a variety of sources (Word, Excel, web pages, Google Sheets, etc.) directly into the editor while preserving structure and formatting
  • Link Validator (2025 – Redesigned): a fully revamped system featuring a dedicated Link Validation section, Google Sheets export, Outlook email integration, and archive management, streamlining workflow and enhancing user focus

Core functionality

  • Syntax cleanup
    removes proprietary Microsoft markup, deprecated attributes, extra spaces, line breaks, and empty tags for cleaner, standards-compliant HTML
  • Dynamic language detection
    applies language-specific formatting to English or French content
  • Content structuring
    builds table of contents, corrects malformed structural elements, encodes images and bilingual footnotes, adds semantic section tags, and formats definition lists
  • Table accessibility

This tool easily synchronizes two HTML tables by transferring only the updated content from one table to another, while preserving the original formatting and structure.

Purpose & Value

  • Update tables safely – designed for situations where tables need to be updated without breaking existing formatting
  • Avoid errors – prevents blindly copying everything and accidentally overwriting correct data
  • Save time – syncs only the content that has changed, reducing manual work
  • Preserve formatting – keeps table structure and styles intact so users don’t need to reapply formatting
  • Visual difference tracking – highlights changes between tables, making updates easier to review and errors easier to catch
  • Simplify table comparison – makes comparing and updating multiple tables fast, simple, and safe

Key Features

  • Dual Table Interface
    two side-by-side panels with textareas for pasting HTML table code, live previews that update as you type, and automatic highlighting of differences between tables
  • Synchronization Controls
    Buttons to copy cell data, headers, captions, or all elements from the second table to the first
  • Table Management
    reset tables to their original state, clear both textareas, and copy the modified table’s HTML to the clipboard
  • Smart Features
    automatically removes extra paragraph tags and whitespace, checks table structure before syncing, and highlights updates with color-coded feedback in real time
